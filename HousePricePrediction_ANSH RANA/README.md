# House Price Prediction using Machine Learning

## Project Overview

This project uses Machine Learning techniques to predict house prices based on various property features such as area, number of bedrooms, bathrooms, parking availability, furnishing status, and other amenities.

## Dataset

The project uses the Housing.csv dataset containing residential property information and sale prices.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Feature Engineering
4. Linear Regression Model
5. Random Forest Regressor Model
6. Model Evaluation
7. Data Visualization
8. Business Insights and Recommendations

## Model Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

## Visualizations

* House Price Distribution
* Correlation Heatmap
* Actual vs Predicted Prices

## Results

The Random Forest Regressor achieved better performance than Linear Regression, demonstrating the presence of non-linear relationships within housing data.
